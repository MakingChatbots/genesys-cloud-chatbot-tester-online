About

This website is a free and open-source tool to allow chatbot creators/testers to simulate conversations for their Genesys Cloud based chatbots.

Since this tool uses Genesys Cloud's Deployment ID, it can be used to test any Genesys Cloud chatbot integration that uses Inbound Message flows, such as:

  • Genesys Digital Bot Flows
  • Genesys Dialog Engine Bot Flows
  • Genesys Bot Connector integration
  • WhatsApp
  • Amazon Lex V1 / V2
  • Google Cloud Dialogflow ES / CX

How does it work?

This leverages Genesys Cloud's official Web Messenger Guest API to simulate a user connecting to a Deployment ID and sending/receiving messages.

You can see the code here: https://github.com/MakingChatbots/genesys-cloud-chatbot-tester-online
